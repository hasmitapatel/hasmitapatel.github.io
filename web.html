Web Design

Trees for Canterbury

This website concept is for an non-profit organisation called "Trees for Canterbury". The brief was to create an effective 'call to action' design to engage the consumer to take action for the cause. Because this was an environmental organisation I used a nature colour scheme consisting of greens and browns with somes textures but also added whites to keep the site looking light and clean. This site was designed with purely HTML and CSS.

Online Store

For this assignment we had to create an online store using a content management system. For this website I used Wordpress and created a child theme using Storefront and the Woocommerce plugin. I edited the child theme through a combination of using PHP and Wordpress.

Food Truck Garage

This assignment brief was to create a cuisine website using Javascript and I chose to create a concept website for Food Truck Garage. In this website I managed to incorporate animation using altering the SVG code of the graphics I had created. I also included a scroll progress bar using Javascript and animated in Photoshop.
